# A Ilha Proibida

## Descrição do enredo

**A Ilha Proibida** é um jogo RPG interativo que utiliza apenas linhas de comando no Swift (Command Line Tool).
O jogo é um RPG Fantasy que se baseia em exploraçao, e coleta de itens. 
No jogo, você controla um jovem guerreiro que, após uma emboscada, tem seu barco destruído e acaba preso em uma ilha dominada por magia negra.

Seu objetivo é explorar quatro locais diferentes da ilha, coletar **4 artefatos mágicos**, necessarios para reconstruir um portal antigo, podendo assim, retornar ao **Reino Teruel** e alertar o Rei sobre a emboscada.

O jogo é simples, nao possui sistema de combate, totalmente baseado em menus, estruturas condicionais e eventos narrativos aleatórios.


## Descrição do projeto

Após a introdução, o jogador é perguntado seu nome e apresentado ao menu principal, onde tera acesso a todas a ferramentas do jogo.

As opçoes do menu são:
  "1- Explorar a ilha"
  "2- Ver inventário"
  "3- Ver seu objetivo principal"
  "4- Sair do jogo"

    1- Ao selecionar a opção 1, o jogador é enviado a outro menu perguntando qual local da ilha ele deseja explorar.
      "1- Floresta Sombria"
      "2- Montanhas do Desespero"
      "3- Caverna Amaldiçoada"
      "4- Ruínas do Portal"
      "5- Voltar"
            Ao escolher uma das opções, o jogador é apresentado a um evento aleatorio, podendo encontrar algum artefato magico necessario para reconstruir o portal, um encontro misterioso, ou nada.

    2- Ao selecionar a opção 2, o jogador consulta o inventario, e é mostrado na tela se ele possui algum artefato magico, e quantos faltam para reconstruir o portal.

    3- Ao selecionar a opção 3, o jogador consulta seu objetivo, bem como uma lista de artefatos que faltam para reconstruir o portal.

    4- Ao selecionar a opção 4, o jogo apresenta uma mensagem de saida e termina.
